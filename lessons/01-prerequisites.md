# Lesson 01: Prerequisites

In this tutorial series we will be creating a bubble popping browser game. Before we begin we will need to install the development tools we will be using:
01. install Visual Studio Code from https://code.visualstudio.com/download
02. install Node:
    * Windows: visit https://nodejs.org/en/download
    * Linux: install/update via your package manager
    * MacOS: visit https://nodejs.org/en/download
